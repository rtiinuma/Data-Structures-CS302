Copy Constructor for the binary seach tree

Copy constructor sets root to NULL. The overloaded assignment operator is then called to create a copy of the given binary seach tree.

Parameters:
sourceGiven binary seach tree to be copied.
Precondition:
  1. Function must be called with a valid binary seach tree as the parameter.
Postcondition:
  1. Tree will be created.
  2. Tree will be updated to be a deep copy of the given source binary seach tree.

  1. Tree will have nodes deleted.
  2. Root will be pointing to null.

Overloaded Assignment Operator

Function overloads assignment operator so that one binary seach tree may be assigned to another explicitly. First clears the current tree if necessary and then copies over given tree.

Parameters:
[in]sourceGiven binary seach tree to be copied.
Precondition:
  1. Binary seach tree must be declared.
  2. Source must be valid binary seach tree.
Postcondition:
  1. Tree will become a deep copy of given source tree.
Returns:
Updated tree object.
